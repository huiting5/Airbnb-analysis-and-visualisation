**Airbnb Open Data** from Kaggle: https://www.kaggle.com/datasets/arianazmoudeh/airbnbopendata 
House Rules: A set of rules or expectations set by the host for the behavior of guests inside the house.

Neighbourhood Group:  The location of the house is mainly in New York City. The dataset includes the counties of Bronx, Brooklyn, Manhattan, Queens, and Staten Island.

Price: Daily price in local currency. Here is USD.

lat & long: Uses the World Geodetic System(WGS84) projection for latitude and longitude

Room Type: Types of rooms. There are entire homes, hotel rooms, private rooms, and shared rooms.
